# GitHub Profile Finder

A web app to search any GitHub user and view their profile details instantly.

## Features

- Search any GitHub username
- View profile picture, name, and bio
- See public repos, followers, and following count
- Direct link to GitHub profile
- Error handling for invalid usernames
- Press Enter key to search

## Tech Stack

- HTML
- CSS
- JavaScript (Fetch API, async/await)

## How to Use

1. Open `github-finder.html` in your browser
2. Type any GitHub username in the search box
3. Click Search or press Enter
4. View the profile details!

## API Used

[GitHub REST API](https://api.github.com/users/{username})
